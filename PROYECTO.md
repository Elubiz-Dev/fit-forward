# Proyecto: Fit-Go
Aplicación móvil de fitness y entrenamiento físico.

## Stack Tecnológico
- Frontend: React Native con Expo (TypeScript)
- Base de datos y Autenticación: Supabase
- Estilos: NativeWind / Tailwind CSS

## Módulos y Estructura Actual
- /app: Pantallas principales (Login, Inicio, Perfil, Historial)
- /components: Tarjetas de ejercicios, gráficos de progreso, botones custom
- /supabase: Rutas de consultas, tablas de usuarios y registro de peso

## Características actuales instaladas
- Registro de peso (75kg actual) e historial.
- Mapa muscular interactivo básico.
- Base de datos de alimentos y carnes.

## Lo que busco lograr
- Crear una app más fluida que Fitia o MyFitnessPal.
- Sistema de entrenamiento rápido y gamificado.
